== Description ==

Super simple backup taker for wp-content folder and the database. Super useful for developers or before plugin/theme update. 

Activate the plugin, go to the Simple Copy menu and take as much zip archives of your themes plugins and database as you want.

Files are saved under wp-contet/ directory.
